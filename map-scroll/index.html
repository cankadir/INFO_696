A simple scrolly story template for maps

This site is a template for using a map as a scrolly story. It uses the enter-view library to trigger events when the user scrolls to a certain point in the page. The map is created using leaflet. The data in the story is entered into JS manually since it is small enough. If you have a lot of data, you can use a CSV or JSON file to import the data using a fetch function.
